■説明事項
　・基本的な事
　　わおん（Twitter：＠waon_0717）が製作しているゴーストです。（ゴースト内の作者名は「lupinus」になっています）
　　栞は里々です。

　・特徴
　　とにかくランダムにランダムトークをします。
　　背景、性格、物語、その他もろもろの設定はありません。
　　もう何パターンのトークが存在するのか作者も把握できていません。
　　羽根の配色パターンはなぜか27^6=387420489通りくらいあります。
　　トーク内容のサンプルは作者ついったーとかに多分あります。

　・基本操作
　　ホイールクリックでメニューが出ます。
　　ひたすら胡乱なことを言っているのを眺めます。
　　たまに胡乱なものを拾ってきます。
　　拾って来たものはメニューから見せてもらうことができます。特に意味はありません。
　　羽根の色はランダムに変わったりします。設定から配色を固定したり、着せ替えから好きな色に変更することも可能です。

　・注意点
　　「SSP」および「Androidの伺か」上で動作確認をしています。それ以外のベースウェア上での動作については全く保証できません。 
　　ランダム生成の結果によってはものすごく酷いことなどを言う可能性があります。あらかじめご了承ください。
　　パスワードは「HarmoniaMaris」です。全角/半角、大文字/小文字は問いません

　・現存する不具合とか困った仕様とか
　　☆拾ったものが延々とセーブデータに記録されるので長期的に面倒なことになる可能性がある（未確認）（とりあえず400個くらいまではなんの問題もない）
　　　→セーブデータを開いて＄持ち物の中身を手動で削除/コピーしてもらう必要があるかもしれない
